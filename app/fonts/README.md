# Custom Fonts Directory

Place your custom font files here.

## Supported Formats
- `.woff2` (recommended - best compression)
- `.woff` 
- `.ttf` (TrueType)
- `.otf` (OpenType)

## How to Add Your Font

1. Copy your font file(s) to this directory
2. Update `app/layout.tsx` with your font name and file path
3. The font will be automatically loaded and optimized by Next.js

## Example Font File Names
- `MyCustomFont-Regular.woff2`
- `MyCustomFont-Bold.woff2`
- `MyCustomFont-Italic.woff2`

## Font Weight Values
- 100 - Thin
- 200 - Extra Light
- 300 - Light
- 400 - Regular/Normal
- 500 - Medium
- 600 - Semi Bold
- 700 - Bold
- 800 - Extra Bold
- 900 - Black
